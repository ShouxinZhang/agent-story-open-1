# 世界背景 (World Background)

## 1. 时代与宇宙观
- **时间线**：龙历2126年 / 星际联邦纪元第三十七年。
- **背景概述**：人类已全面迈入星际航行与极高维度的“基因觉醒”时代。从银河系猎户臂到各大星系（如鲸鱼星系）之间，可以通过多次空间折叠进行超光速跃迁，星际间有着密集的港口航道（如龙国东部最大的临海星港），连接着数百个殖民星与资源星。
- **多元宇宙设定**：在当前宇宙（星际联邦）之外，证实存在无数个具有独立规则体系的平行/高维宇宙（如洪荒世界、异能之王世界、仙侠世界等）。不同世界的法则（神力、异能、灵气等）能通过高阶系统转译映射到本宇宙，表现为精神力变异或提取暗物质辐射（宇宙背景能量）。

## 2. 核心力量体系：基因血脉
全人类经历过“大觉醒”，基因血脉力量成为社会地位、战力评估的核心指标。
- 等级被严格划分为 F、E、D、C（普通至基层）、B（战术门槛/顶尖学府线）、A（战略级精英，掌握领域或规则雏形）、S（行星级），以及传说中的 SS、SSS 与 X 级（宇宙神话级）。
- 能力表现涵盖元素干涉、精神感知、肉体超体等。部分极优秀且拥有适配高阶功法的觉醒者，可以将古老体系（如修仙的呼吸法）隐蔽地融合进基因力量的发展中。

## 3. 主要势力与社会结构
- **星际联邦 (Star Federation)**：横跨星系的最高人类联合政权。下设海防部、网络安全部门等正规军政机构。负责对抗各大危星星域的异兽狂潮，主导“联邦高校联合维和实训”来提拔年轻战将。
- **龙国 (Dragon Nation)**：星际联邦的核心成员国之一。内部资源和权利很大程度掌控在“四大基因世家”手中，例如“临海苏家”（掌握数十颗资源星开采权）和“京都叶家”。
- **宇宙异兽 (Alien Beasts)**：宇宙中普遍存在的敌对或中立超自然生物，等级从F级别（如无害的翡翠滑翔鳐）到S级以上（深海巨兽）皆有，高阶异兽具备引发星球级灾难的能力。

## 4. 特殊星球档案：海神星 (Neptune Star)
第七维和实训小队当前任务驻地。
- **坐标与规格**：位于鲸鱼星系。半径约地球4倍，表面积为地球16倍，引力 1.18G。拥有两颗卫星（双月），大气高氧，伴有特殊植物花粉及微量高纯度背景辐射（暗物质+基因矿脉辐射）。
- **环境组成**：73%被海洋覆盖。陆地多为超大型群岛和漂浮大陆。
- **原住民与异兽**：拥有约900亿类人原住民“海神族”（具备蓝肤、鳍脊结构，F-D级初级觉醒，科技处于类似工业革命时期）；南部深海海沟频发S级高等异兽暴动，北部翡翠群岛协防区当前为实训缓冲区，但异兽正逐渐北延。
