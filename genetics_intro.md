# 星际联邦基因血脉系统介绍 (Intro)

## 1. 体系概述
在星际联邦纪元（龙历2126年），全人类在迈入星际文明的过程中，经历了被称为“大觉醒”的基因突变。基因血脉不仅仅是身体素质的强化，更是涉及元素控制、精神感知、空间干涉等超自然能力的觉醒。

## 2. 等级划分
基因血脉按潜力和强度分为七个大阶级，每个大阶级内细分为五个小等级（Lv.1 - Lv.5）。

*   **F - E 级 (平庸血脉)**
    *   **占比**：约占联邦总人口的60%。
    *   **特征**：基础身体素质强化，如抗衰老、微小的力量或视觉提升。适合普通工种和基础岗。
*   **D - C 级 (精英血脉)**
    *   **占比**：约占联邦总人口的30%。
    *   **特征**：具备初步的特异显现，如微弱的火焰控制、肉体局部硬化等。星际舰队基层士兵和普通学者的普遍等级。
*   **B 级 (战术血脉)**
    *   **占比**：约占联邦总人口的9%。
    *   **特征**：能力具有实战价值，可以改变局部战局。能力显现明显，破坏力或辅助功能较强。能进入顶尖学府（如信息工业大学）的最低门槛。
*   **A 级 (战略血脉)**
    *   **占比**：极少（不到1%）。
    *   **特征**：一人即成军。能力往往涉及规则、领域或高阶物理法则的干涉。不仅取决于先天基因，更需要海量的资源进行开发。联邦各大世家和高层的中坚力量。
*   **S 级 (行星级血脉)**
    *   **占比**：凤毛麟角。
    *   **特征**：已经具备单体对抗星际战舰、镇压一片战区的能力，是足以左右行星级战局的战略个体。每一次 S 级及以上基因的觉醒，都会被星际联邦最高议会立档监管。
*   **SS 级 (恒星级血脉)**
    *   **占比**：近乎绝迹。
    *   **特征**：能力往往触及引力、时间、空间或大气海洋等宏观规则，足以摧毁舰队编组、瘫痪大型殖民区，甚至改写一颗星球的生态与战争格局。
*   **SSS 级 (恒星级巅峰血脉)**
    *   **占比**：只存在于最高机密档案与少数传说之中。
    *   **特征**：已接近文明灾厄级个体，能够以单人之力撼动整颗星球的防御体系，或在特定环境中展现堪比天灾的主权级支配力。
*   **X 级 (宇宙级血脉/神话级)**
    *   **占比**：传说。
    *   **特征**：足以改写星图、扭曲时间与空间维度的禁忌存在。目前仅存在于高层绝密档案和虚无缥缈的神话描述中。

## 3. 等级提升 (Lv.1 - Lv.5)
每个大阶（如B级）的觉醒者，可以通过锻炼、基因药剂、实战极限突破（经验值）来提升当前阶级的小等级。
*   **Lv.1**: 觉醒初期，能力不稳定。
*   **Lv.3**: 熟练掌握，能进行高强度的持续输出或控制。
*   **Lv.5**: 大圆满阶段，部分天才可能在此时触碰到“阶级跃迁”的壁垒（如从B级Lv.5突破至A级Lv.1，但概率极低）。

## 4. 体系分类
基因血脉一般分为三大类：
1.  **元素干涉系**：操控地火水风、雷电、光暗等自然元素。
2.  **肉体超体系**：绝对力量、神速、自愈、肉体金属化、巨兽化等。
3.  **精神感知系**：念动力、心电感应、物质结构透视、微观干涉、预知等。
